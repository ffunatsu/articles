---
title: 260504 GitHub Copilot の利用停止
---
あまりAIネタを書きたくはないのだけれど、[GitHub Copilotの従量課金制への移行](https://github.blog/jp/2026-04-28-github-copilot-is-moving-to-usage-based-billing/) は個人的に大きなニュースだったので一応言及しておこうかと。

タイトルにある通り、自分はこれを機に利用停止することにした。元々あまり使っていなかったので一時措置でProへの申込みができなくなったのもあるし、たまに使っていたくらいなので利用停止しても支障が特にないというのもある。

他の代替手段についても、元々生成AI自体あまり使っていなかったので、軽く検討はしたものの金額が高かったのもあり今のまま使わない方針。

この件でどうやらGitHub Copilotを完全に利用停止することはできなくなった様子で、元々そうだったようなのだけれど、Copilotアイコン類をエディタ上で非表示にしたり拡張機能をアンインストールすることくらいしかできない様子。（拡張機能をアンインストールしても、さらに非表示を選択しないと死神のようにアイコン類は復活し続ける様子。）

この辺は、Kindle Paperwhiteの電源オフ機能がなくなったのと同じで、やっぱり大企業だなと思う。多くのユーザが最近codeberg等に移行するのも頷けるし、このブログもなし崩し的にGitHubに置いているけれど、そろそろホスト先も変更しても良いかもしれない。

自分はMicrosoftとかGoogleについては、生成AI誕生以前はどちらかといえば印象は良かったのだけれど、生成AI以降のスタンスは正直あまり好きになれなくて、検索エンジンも最近DuckDuckGoを中心に利用していたりする。

これ関連の話題はあまり書いていて良い気持ちになれないので、やはり今後は言及をしないでおくくらいで良い気もしてきた。
